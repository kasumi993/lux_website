== Description ==
Kata App is Free WordPress Theme easy use and user friendly, for all WordPress users, Kata App supported 3 widget area for footer and 2 side bar widget area for blog and single pages.
Kata App created based Elementor WordPress plugin so you can install and use elementor when Kata App WordPress theme is activated in your WordPress site without any warriors.

== Installation ==
1. After Installing WordPress go to WordPress dashboard and navigate to Appearance > themes
2. Find Add new Button and click on there
3. type "Kata App" In search box and find "Kata App" WordPress Theme, Then click on the install button after finish install progress click on activate button
4. Navigate to Appearance > Customize in your admin panel and customize to your needs.
5. Enjoy

**Does this theme support any Elementor?**

Kata App is completely compatible with Elementor.
